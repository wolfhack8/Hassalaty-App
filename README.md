# حصالتي 

منصة عربية لتعليم الأطفال الوعي المالي، مبنية على روح مشروع «نقد» وموسعة بلوحة أولياء الأمور، اللعب بالنقاط، المكافآت والذكاء الاصطناعي.

## التشغيل والنشر

1. انسخ `.env.example` إلى `.env` وأدخل رابط PostgreSQL و`AUTH_SECRET`.
2. شغّل `npm install` ثم `npm run db:push` و`npm run db:seed`.
3. شغّل `npm run dev`، أو ارفع المشروع إلى Vercel وأضف متغيرات البيئة نفسها.

تفعيل تسجيل Google وApple يتطلب مفاتيحهما في متغيرات البيئة. الحساب التجريبي: اضغط «دخول الحساب التجريبي» بعد تشغيل البذور.

## البنية

- `src/app/parent`: لوحة ولي الأمر وتقرير الطفل.
- `src/app/child`: لوحة الطفل، اللعبة، المتجر، والمساعد الذكي.
- `src/app/api/ai`: تحليل الوالد والمساعد والتحديات الذكية.
- `src/app/api/children/[id]/points`: حفظ نقاط اللعبة بمحدد يومي.
- `prisma/schema.prisma`: الأدوار والعلاقات والمعاملات والأهداف والمكافآت والتقارير.
