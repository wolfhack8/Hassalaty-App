# 🪙 منصة حصالتي - Hassalaty App 

> **النسخة التجريبية (MVP) المطورة خلال منافسات الهاكاثون الرقمي للتقنية المالية ٢٠٢٦.**

منصة **حصـالتي** هي منظومة تقنية مالية (FinTech) عائلية متكاملة، تهدف إلى إعادة صياغة الوعي المالي وتنمية مهارات التخطيط الاستراتيجي لدى جيل  في ظل التحول الرقمي والمجتمع اللانقدي.

---

## ✨ المميزات الرئيسية والابتكارات
* **محاكاة الصناديق الثلاثة الذكية:** تقسيم مبرمج وتلقائي لمصروف الأبناء إلى (صندوق إنفاق، صندوق ادخار للأهداف، وصندوق عطاء ومسؤولية مجتمعية).
* **لوحة التحكم العائلية (الآباء):** واجهة متقدمة تتيح للآباء أتمتة المصاريف، وإسناد المهام المنزلية، ومراقبة السلوك المالي للأبناء برقابة صامتة ومرنة.
* **كوتش الذكاء الاصطناعي الاستباقي (كوتش ذُخر):** مستشار ذكي مدمج يحلل وتيرة صرف الابن لحظياً ويتدخل تربوياً لتنبيهه لحماية ميزانيته من النفاد المبكر.
* **نظام التلعيب المالي (Financial Gamification):** ربط الجهد بالمال من خلال تمكين الطفل من تنفيذ المهام وتحويل المكافآت مباشرة لصندوق أهدافه الادخارية.

---

## 🛠️ المكدس التقني (Tech Stack)
* **Framework:** React.js (Vite Bundle)
* **Styling:** Tailwind CSS + PostCSS
* **Icons:** React Icons (FontAwesome)
* **Architecture:** Component-driven Responsive Design
* **Deployment:** Hosted live on Vercel

---

## 🚀 طريقة التشغيل والتركيب المحلي

تأكد من وجود بيئة `Node.js` مثبتة على جهازك، ثم نفذ الأوامر التالية بالتتابع داخل شاشة موجه الأوامر (Terminal):

```bash
# 1. تثبيت كافة الحزم والاعتماديات البرمجية المعتمدة للهاكاثون
npm install

# 2. تشغيل سيرفر المطورين المحلي للمنصة
npm run dev
